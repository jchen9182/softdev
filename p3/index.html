SHELL SCRIPTING

What is the Kernel?

The kernel is a computer program that controls everything in the system.
It handles interactions between the hardware and software and is basically the core of the OS.
For example, it might manage files, memory, or I/O.

What is the Shell?

The shell is a program that lets the user interact with the kernel.
It's accessed from the command line or terminal.
Some examples are bash, zsh, and csh.

BASH Shell Scripts

A shell script is a file that contains commands that the shell runs.
They allow you to automate things and add your own commands to the shell.

Pros

It's easy to write short scripts.
The commands you type are the same as the ones used in bash.
Piping is very simple to do. You can read more about piping here.

Cons

Execution time is slow.
Shell scripts aren't well-suited for long and complex tasks.

Notes

Make sure your script is executable by doing chmod +x [path to script].

Your script can be saved with .sh or no extension at all.

Your script must always have #!/bin/bash at the beginning of the file.
It lets the system know that this is a bash script.

To execute your script, there are many ways such as ./[script] or bash [script].
